# HSE Franca – Site Institucional (Segurança do Trabalho & Perícia)

Um site estático, leve e responsivo para **HSE Franca**.
Você pode **substituir facilmente as imagens** na pasta `assets/img/` pelos seus arquivos.

## Como editar
- **Logo**: troque `assets/img/logo.svg` por sua marca (mantenha o mesmo nome para não mudar o HTML).
- **Imagem do Hero**: substitua `assets/img/hero.jpg`.
- **Imagens dos serviços**: substitua `assets/img/servico-*.jpg`.
- **Cores**: edite as variáveis no topo de `assets/css/style.css`.
- **Texto/Serviços**: o HTML está em `index.html`. Você também encontra a lista de serviços (cards) na seção "Serviços".
- **Links de contato**: altere telefone, e-mail e endereço no rodapé e na seção Contato.

## Estrutura
```
hse-franca-site/
├─ index.html
├─ assets/
│  ├─ css/
│  │  └─ style.css
│  ├─ js/
│  │  └─ main.js
│  └─ img/
│     ├─ logo.svg
│     ├─ hero.jpg
│     ├─ servico-gro.jpg
│     ├─ servico-ltc a t.jpg
│     ├─ servico-insalubridade.jpg
│     ├─ servico-periculosidade.jpg
│     ├─ servico-aet.jpg
│     ├─ servico-nrs.jpg
│     ├─ servico-pericia.jpg
│     └─ placeholder.svg
```
> Obs.: as imagens `*.jpg` são apenas **placeholders**. Pode manter o nome e substituir o arquivo por outro JPG/PNG.

## Hospedagem
